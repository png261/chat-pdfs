## Setup 
### Install dependencies
```
pip install -r requirements.txt
```
### Add .env
```
LANGSMITH_TRACING=
LANGSMITH_ENDPOINT=
LANGSMITH_API_KEY=
LANGSMITH_PROJECT=
GOOGLE_API_KEY=
TAVILY_API_KEY=
```

## Run
```
streamlit run app.py
```
